Detailed Description

template<typename T, unsigned int N>
class LUMatrix< T, N >

Identical class to LUMatrixGPU, except this class is a pure C++ class (rather than a Cuda class) so that it only runs on the CPU. See LUMatrixGPU for more detailed documentation and a usage example.
