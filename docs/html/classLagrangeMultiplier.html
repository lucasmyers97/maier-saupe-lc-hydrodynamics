Detailed Description

template<int order, int space_dim = 3>
class LagrangeMultiplier< order, space_dim >

Takes in a \(Q\)-tensor, outputs corresponding \(\Lambda\) value using the Newton-Rhapson method, where \(\Lambda\) is given by:

\begin{equation} Q_{ij} = \frac{\partial \log Z}{\partial \Lambda_{ij}} - \frac13 \delta_{ij} \end{equation}

with

\begin{equation} Z[\Lambda] = \int_{S^2} \exp[\Lambda_{ij} \xi_i \xi_j] \: d \xi \end{equation}

A typical use-case of this class is as follows:

#include <deal.II/lac/vector.h>
#include <iostream>
const int order = 590;
double alpha = 1.0;
double tol = 1e-12;
int max_iters = 12;
dealii::Vector<double> Q({6.0 / 10.0, 0.0, 0.0, -3.0 / 10.0, 0.0});
dealii::Vector<double> Lambda();
Lambda.reinit(5);
LagrangeMultiplier<order> lm(alpha, tol, max_iters);
lm.invertQ(Q);
lm.returnLambda(Lambda);
std::cout << Lambda << std::endl;

Constructor specifies Newton-Rhapson parameters (step size, error tolerance for convergence, max number of iterations before failure).

Parameters
[in]alpha_Gives step size for Newton-Rhapson method: \(\Lambda_i^{n + 1} = \Lambda_i^n + \alpha \delta \Lambda_i^n\)
[in]tol_Gives error tolerance for Newton-Rhapson method to terminate: \(|R_m(\Lambda^n)| < \epsilon \) where \(R_m\) is the residual, and \( \epsilon\) is the tolerance.
[in]max_iter_Maximum number of iterations for Newton-Rhapson method – after this many iterations, the method terminates and inverted flag is set to false.

Calculate one term in the lebedev quadrature sum corresponding to the following integral: \(\int_{S^2} \xi_{i(m)} \xi_{j(m)} \exp[\Lambda_{kl} \xi_k \xi_l] d\xi\) where \(i(m)\) is the row index in the Q-tensor corresponding to the \(m\)th entry of the Q-vector, and \(j(m)\) is the column index in the Q-tensor corresponding to the \(m\)th entry of the Q-vector. The quadrature sum is calculated one term at a time to avoid having to calculate the same \(\exp[\Lambda_{kl} \xi_k \xi_l]\) multiple times.

Parameters
[in]exp_lambdaHolds the value of \(\exp[\Lambda_{kl} \xi_k \xi_l]\) for a particular \(\Lambda\) and Lebedev point.
[in]quad_idxIndexes which term in the quadrature sum we are calculating.
[in]i_mRow index in Q-tensor corresponding to m-th entry of Q-vector
[in]j_mColumn index in Q-tensor corresponding to m-th entry of Q-vector.
