\subsubsection*{template$<$int order, int space\+\_\+dim = 3$>$\newline
class Lagrange\+Multiplier$<$ order, space\+\_\+dim $>$}

Takes in a $Q$-\/tensor, outputs corresponding $\Lambda$ value using the Newton-\/\+Rhapson method, where $\Lambda$ is given by\+: 

\begin{equation} Q_{ij} = \frac{\partial \log Z}{\partial \Lambda_{ij}} - \frac13 \delta_{ij} \end{equation} with \begin{equation} Z[\Lambda] = \int_{S^2} \exp[\Lambda_{ij} \xi_i \xi_j] \: d \xi \end{equation} A typical use-\/case of this class is as follows\+:
