Reads Lebedev coordinates and weights in from global memory to shared memory for any given block of threads. For testing, it outputs each set of Lebedev coordinates and weights (that is, corresponding to each block) in its place in a new global array. This will be copied to the host and checked to make sure every block\textquotesingle{}s shared memory has a copy of the Lebedev data.
